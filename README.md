# Machine Learning Based Alpha Generation for NIFTY 100

## Project Summary
• Built LightGBM model to predict cross-sectional alpha using financial features and news data  
• Evaluated signals using Information Coefficient and long–short portfolio simulations
• Constructed portfolio of top 10 stocks with monthly rebalancing
• Evaluated the portfolio by using metrics such as CAGR > 20%, Sharpe ratio > 1, Sortino ratio ~1.5, and Max. Drawdown ~20%
